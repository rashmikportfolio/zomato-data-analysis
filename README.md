# zomato-data-analysis
## Project Overview
Analyzed the Zomato restaurant dataset using Python and performed exploratory data analysis.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Most restaurants fall under Dining and Delivery categories.
- Distribution of ratings was analyzed.
- Online ordering and ratings relationship was studied.
- Cost for two and restaurant categories were explored.
